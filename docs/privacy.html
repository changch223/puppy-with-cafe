🐶☕ Puppy With Cafe プライバシーポリシー

最終更新日: 2026年7月5日

Puppy With Cafe（以下「本アプリ」）は、利用者のプライバシーを最大限尊重して設計されています。

1. 収集しない情報

  • アカウント登録はありません。氏名・メールアドレス等の個人情報を収集しません。
  • 広告・トラッキング（追跡）は一切行いません。解析SDK・広告SDKを含みません。

2. 位置情報について

  • 位置情報は「周辺の犬同伴可カフェを探す」機能のためだけに、利用者の許可を得たうえで（使用中のみ）取得します。
  • 位置情報の処理はすべて端末内で完結し、外部に送信・保存されることはありません。
  • 位置情報を許可しなくても、地域を手動で選んで利用できます。

3. 通信について

  • 本アプリは、カフェ情報データ（cafes.json）を取得するためにのみ通信を行います。この通信に利用者の個人情報・位置情報は含まれません。

4. 誤り報告（外部フォーム）

  • カフェ情報の誤り報告には Google フォームを利用します。フォームの利用には Google のプライバシーポリシーが適用されます。フォームで任意に入力された内容は、掲載情報の修正のためだけに利用します。

5. 外部リンク

  • 本アプリには各店舗の公式サイト・SNS等へのリンクが含まれます。リンク先での情報の取り扱いは各サイトのポリシーに従います。

6. 掲載情報について

  • 掲載しているカフェ情報（犬同伴可否・営業時間等）は、出典と確認日を明示した収集時点の情報です。最新の情報は各店舗の公式情報をご確認ください。

7. 改定・お問い合わせ

  • 本ポリシーを改定する場合は、本ページで告知します。
  • お問い合わせ: GitHub リポジトリの Issue までお願いします。
